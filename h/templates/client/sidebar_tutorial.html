How to get started

  1. To create an annotation, select text and click the  button.

  2. To add a note to the page you are viewing, click the  button.

  3. To add a note to the page you are viewing, click the  button.

  4. To create a highlight, select text and click the  button.

  5. To reply to an annotation, click the  Reply link.

  6. To share an annotated page, click the  button at the top.

  7. To create a private group, select Public, open the dropdown, click + New group.
