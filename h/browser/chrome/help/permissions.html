Access denied

Maybe you are trying to access a local document, using the file:/// protocol, but forgot to grant the required permissions to the Hypothes.is extension?

If that's the case, here is how to allow the Hypothes.is extension to access file URLs:

  1. Go to chrome://extensions (Sorry, we can't make this clickable; you will need to type or copy this.)
  2. Check this checkbox called "Allow access to file URLs", as shown here:

That still won't make local HTML documents useable (for now), but at least local PDF documents can be annotated this way.
